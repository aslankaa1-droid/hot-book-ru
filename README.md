# Hot Book — лендинг сайта

Премиум-лендинг приложения **Hot Book** — личного книжного соавтора с локальным ИИ, который живёт в устройстве автора (Windows · Android · iOS).

## Состав

| Файл | Назначение |
|------|-----------|
| `index.html` | Весь сайт в одном файле: разметка, стили и логика. Ассеты (логотип, иконки, фавикон) встроены как inline-SVG — внешних картинок нет. |
| `.nojekyll` | Отключает обработку Jekyll на GitHub Pages (нужно, чтобы файлы отдавались как есть). |
| `README.md` | Этот файл. |

Шрифты подключаются с Google Fonts (Fraunces, Hanken Grotesk, Spectral, Noto Kufi/Naskh Arabic). Других внешних зависимостей нет.

- **Как это работает** — демо рабочего экрана с агентом «Атос» + панель «Библия книги».
- **Сравнение** с облачными ИИ-редакторами и обычными редакторами.
- **Платформы** — Windows / Android / iOS.
- **FAQ** — 6 вопросов.
- **Контакты** — mailto / tel / Telegram / WhatsApp + дисклеймер 152-ФЗ.

## Соответствие 152-ФЗ

На сайте **нет** форм обратной связи, **нет** Яндекс.Метрики и Google Analytics, персональные данные не собираются. Связь — только по контактам (email / телефон / мессенджеры) по инициативе посетителя. В подвале — соответствующий дисклеймер.

> Контакты на странице: email `aslankaa@yandex.ru` (канон), а также Telegram, WhatsApp и телефон. Ссылки на мессенджеры и номер телефона — заглушки (`t.me/aslankaa`, `wa.me/79000000000`, `tel:+79000000000`); при публикации подставить реальные значения заказчика.

## Локальный просмотр

Достаточно открыть `index.html` в браузере двойным щелчком. Для корректной работы Google Fonts нужен интернет (без него подхватятся системные fallback-засечки, но это не канон — для финального вида интернет желателен).

## Деплой на GitHub Pages

1. Создать репозиторий (например, `hot-book-site`) на аккаунте `aslankaa1-droid`.
2. Залить содержимое папки `08_Сайт\` в корень ветки `main` (файлы `index.html`, `.nojekyll`, `README.md`).
3. В **Settings → Pages** выбрать ветку `main`, папку `/ (root)`, сохранить.
4. Через 1–2 минуты сайт доступен по `https://aslankaa1-droid.github.io/hot-book-site/`.
5. Для своего домена — добавить файл `CNAME` с доменом и настроить DNS на reg.ru (A-записи на IP GitHub Pages), дождаться выпуска SSL.

Перед публичным деплоем заменить заглушки контактов (Telegram, WhatsApp, телефон) на реальные.
